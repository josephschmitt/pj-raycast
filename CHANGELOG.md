# PJ Changelog

## [1.0.0] - 2026-02-01

### Added

- Initial release
- List Projects command with search functionality
- Actions: Open in Editor, Finder, Terminal, Copy Path
- Configurable default action preference
- Configurable editor and terminal applications
- Project type tags (git, npm, cargo, go, etc.)
- Keyboard shortcuts for all actions
- Refresh action to re-fetch projects
